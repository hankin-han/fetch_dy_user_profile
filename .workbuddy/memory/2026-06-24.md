## 语音转文字服务（百度长语音识别 API）

放弃 whisper.cpp（需编译 FFI 扩展）和 ONNX 浏览器端方案，改用百度长语音识别 API：

**方案选择**：
- whisper.php → 需要 ext-ffi，宝塔 PHP8.1 未开启
- whisper.cpp → 需编译 + 下载 1.5GB 模型
- 百度长语音 API → 零依赖，15 万次免费，最长 5 小时音频

**新建文件**：
- `demo/whisper_api.php`：PHP 后端，接收音频 → 保存到 uploads/ → 调百度 API（创建任务 + 轮询）→ 返回文本 → 清理临时文件
- `demo/uploads/`：临时音频目录

**修改文件**：
- `demo/txt.html`：删除 transformers.js/pipeline/loadModel（~120 行 JS），改为 fetch POST 到 whisper_api.php

**用户需要填写**：`whisper_api.php` 中的 `BAIDU_API_KEY` 和 `BAIDU_SECRET_KEY`
