使用方法

四步完成数据获取

无需安装任何插件，浏览器控制台直接运行

1

打开用户主页

打开 Chrome 浏览器，确保已登录抖音账号，然后访问目标抖音用户主页（如 https://www.douyin.com/user/...）。

2

打开开发者工具

按键盘 F12 打开浏览器开发者工具，点击顶部标签栏切换到 Console（控制台）。

3

粘贴并执行脚本

点击下方「复制脚本代码」按钮，将代码粘贴到控制台输入框中，按 Enter 回车执行。

4

使用侧边面板

页面右侧弹出侧边抽屉面板，数据自动拉取后所有功能均在面板内操作，支持切换视图、搜索筛选、导出下载。

注意事项

使用前请知悉

必须在登录状态下使用抖音账号，未登录状态下 API 返回数据不完整，可能导致获取失败。
